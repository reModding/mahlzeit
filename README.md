mahlzeit
========

Voraussetzungen
---------------

- Ruby 2.0
  - YAML-Support


Installation
------------

```bash
git clone ...
cd mahlzeit/
cp mahlzeitbot.yml.dist mahlzeitbot.yml
cp mahlzeitbot_cache.yml.dist mahlzeitbot_cache.yml
vim mahlzeitbot.yml
chmod +x mahlzeitbot.rb
./mahlzeitbot.rb
```
